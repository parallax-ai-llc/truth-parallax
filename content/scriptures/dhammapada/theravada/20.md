The Maggavagga's 17 verses on the path present the Noble Eightfold Path as the supreme way to liberation. The Theravada tradition regards this path as the definitive teaching of the Buddha, comprising right view, intention, speech, action, livelihood, effort, mindfulness, and concentration.

Theravada interpretation understands the path as both gradually cultivated and momentarily perfected in the supramundane path consciousness that eradicates defilements. The declaration that "of all paths, the Eightfold Path is best" has canonical authority.
